# Network Engineering Lab

This repository contains networking simulations created using Cisco Packet Tracer.

## Labs Included

### 1. Basic Router Network

* Router configuration
* Static IP addressing
* Basic connectivity testing

### 2. DHCP Network

* Automatic IP assignment
* DHCP configuration on router

### 3. VLAN + Inter-VLAN Routing

* VLAN configuration
* Switch trunking
* Router-on-a-stick
* DHCP for multiple VLANs

## Tools Used

* Cisco Packet Tracer

## Topics Covered

* Router configuration
* Switch configuration
* DHCP
* VLAN segmentation
* Inter-VLAN routing
* Network troubleshooting
